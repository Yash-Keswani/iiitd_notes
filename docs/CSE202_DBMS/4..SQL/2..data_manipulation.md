# Data Manipulation
### Insert
* `INSERT INTO <table> VALUES (<data tuple>)`

### Delete
* Remove all tuples from a realtion.
* `DELETE FROM <table>`

### Drop Table
* `DROP TABLE r`

### Alter
* `ALTER TABLE r ADD <A> <D>`
	* All new cells are assigned `null`
* `ALTER TABLE r DROP <A>` 
	* Drops an attribute from a table
